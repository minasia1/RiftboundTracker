# Riftbound Point Tracker

A mobile app for tracking points in the Riftbound card game. Built with React Native and Expo.

## Features

- **Dual Point Counters** - Two counters (0-99), with the top one rotated 180° for your opponent across the table
- **Champion Backgrounds** - Each player can select their favorite champion from the Origins set
- **Reset Button** - Quickly reset both counters to zero

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn
- Expo CLI (`npm install -g expo-cli`)
- iOS Simulator (Mac) or Android Emulator

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/RiftboundTracker.git

# Navigate to the project
cd RiftboundTracker

# Install dependencies
npm install

# Start the development server
npx expo start
```

### Running the App

- Press `i` to open in iOS Simulator
- Press `a` to open in Android Emulator
- Scan the QR code with Expo Go app on your phone

## Usage

1. Tap **+** or **−** to change the score
2. Tap the **background** to select a different champion
3. Tap **RESET** to set both scores to 0
